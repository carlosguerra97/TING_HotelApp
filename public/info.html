Información para el cliente

Aquí se puede mostrar una pequeña ayuda para el cliente, así como las extensiones de interés a las que puede llamar para beneficiarse de los diferentes servicios que se ofrecen en el hotel. Se ha implementado utilizando service-workers la opción de utilizar la Web app sin conexión, en esta versión solo se muestra un mensaje indicando que no se puede conectar al servidor, pero se podrían implementar algunas funcionalidades sin conexión.
